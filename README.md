# Portfólio de QA Manual - SauceDemo

Este repositório apresenta um projeto prático de testes manuais realizado na aplicação SauceDemo, com organização das atividades no Jira e documentação no GitHub.

## Objetivo

Demonstrar conhecimentos iniciais em Quality Assurance, com foco em testes manuais, validação dos principais fluxos da aplicação e organização das atividades de teste.

## Aplicação testada

[https://www.saucedemo.com/](https://www.saucedemo.com/)

## Ferramentas utilizadas

- Jira
- GitHub
- Navegador web

## Escopo dos testes

Foram realizados testes manuais nas seguintes funcionalidades:

- login
- exibição de produtos após autenticação
- carrinho de compras
- fluxo de checkout

## Organização das atividades

As atividades foram organizadas no Jira em formato Kanban, com acompanhamento dos tickets por funcionalidade testada.

### Tickets criados
- validar funcionalidade de login
- validar exibição de produtos após o login
- validar carrinho de compras
- validar fluxo de checkout

## Resultado da execução

Os cenários planejados foram executados com sucesso e apresentaram o comportamento esperado durante a validação.

Não foram identificados defeitos nos fluxos testados durante esta execução.
